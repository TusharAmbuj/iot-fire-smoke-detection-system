# 🔥 Fire and Smoke Detection System (ICT Workshop Project)

A basic IoT-based Fire and Smoke Detection System developed as part of an ICT workshop project in college. The system detects smoke or abnormal heat levels and triggers an alert to indicate a possible fire hazard.

---

## 📌 Project Overview

Fire accidents can cause severe damage to life and property. Early detection is crucial for preventing disasters. This project demonstrates a simple embedded system that monitors environmental conditions using sensors and activates an alarm when potential fire or smoke is detected.

---

## 🚀 Features

• Real-time fire and smoke detection  
• Automatic alarm triggering  
• Sensor-based monitoring system  
• Low-cost embedded hardware implementation  
• Demonstrates IoT and safety monitoring concepts

---

## 🛠 Technologies & Components Used

**Hardware**

- Arduino / Microcontroller
- MQ-2 Smoke Sensor
- Temperature Sensor (LM35 / DHT11)
- Buzzer
- LED Indicator
- Breadboard
- Jumper Wires
- Power Supply

**Software**

- Arduino IDE
- Embedded C / Arduino Programming

---

## ⚙️ Working Principle

1. The smoke sensor continuously monitors smoke concentration in the air.
2. The temperature sensor measures surrounding heat levels.
3. If the smoke or temperature exceeds a predefined threshold:
   - The system identifies it as a potential fire hazard.
   - A buzzer alarm is triggered.
   - LED indicator turns on to alert nearby users.
